## Gemini API key 발급 과정
1. https://aistudio,google.com 접속
2. 구글 계정 로그인
3. 왼쪽 하단 `Get API key` 선택
4. 우측 상단 `API 키 만들기` 선택
5. 처음의 경우 프로젝트 생성을 해야 합니다. - 아닐 경우에는 기존 프로젝트 선택으로 API key를 발급 받을 수 있습니다.
